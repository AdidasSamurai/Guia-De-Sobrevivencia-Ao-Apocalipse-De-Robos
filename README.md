# Protocolo de Sobrevivência Z-1

Este é um sistema de interface estratégica desenvolvido para auxiliar sobreviventes na identificação e relato de ameaças mecânicas em um cenário pós-apocalíptico.

## Sobre o Projeto
O site foi construído como um exercício de **Desenvolvimento de Interfaces Web**, focando em organização de arquivos, semântica HTML, estilização CSS centralizada e manipulação de mídias locais.

## Tecnologias Utilizadas
* **HTML5**: Estruturação de dados, tabelas e formulários.
* **CSS3**: Estilização industrial personalizada e layout responsivo/centralizado.
* **Mídia Local**: Integração de áudio (MP3), vídeo (MP4) e galeria de imagens.

## Estrutura de Arquivos
O projeto segue uma hierarquia organizada para facilitar a manutenção:
* `/paginas`: Contém o Guia Visual e o Formulário de Contato.
* `/imagem`: Banco de dados visual das ameaças.
* `/audio` e `/video`: Registros de campo recuperados.
* `/icones`: Favicon e elementos de navegação.
* `style.css`: Estilização global do protocolo.

## Funcionalidades
1. **Catálogo de Ameaças**: Galeria de reconhecimento com fotos reais e tabela de comparação de perigo.
2. **Registros de Campo**: Player de vídeo e áudio integrados para estudo de comportamento das máquinas.
3. **Relatório Criptografado**: Formulário completo com validação de dados para envio de novos avistamentos.
